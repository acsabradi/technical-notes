# Előfeltételek
Node és npm telepítése
Ellenőrzés:
```bash
node -v
npm -v
```
# Telepítés
```bash
npm install -g @angular/cli
```
Ellenőrzés:
```bash
ng version
```
# Új projekt
```bash
ng new project-name
```
Standalone projekt:
```bash
ng new standalone-project --standalone
```
# Lokális futtatás
```bash
ng serve
```
